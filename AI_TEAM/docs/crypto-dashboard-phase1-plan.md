# Crypto Dashboard Phase 1 Plan

작성일: 2026-04-01

## 전제

- 저장소에는 이번 기능에 대한 실제 PRD/TRD 문서가 커밋되어 있지 않았다.
- 따라서 현재 대화에서 확인된 범위를 1차 구현 스코프로 사용한다.
- 1차 목표는 `웹에서 코인 종목 목록을 보고, 선택한 종목의 가격 그래프를 조회하는 화면`이다.

## 1차 구현 범위

- 웹 앱 기본 구조 생성
- 코인 종목 목록 패널
- 선택 종목 요약 카드
- 가격 그래프 영역
- 간단한 시간 범위 전환
- 데이터 로딩 / 오류 / 빈 상태 처리
- 이후 자동매매 기능을 붙일 수 있는 데이터 계층 분리

## 제외 범위

- 주문 실행
- 실계좌 인증
- 전략 편집기
- 백테스트
- 알림
- 포트폴리오 / 보유 자산 UI

## 구현 단계

### 1. Foundation

- `web/` 앱 스캐폴드
- React + TypeScript + Vite 기반 개발 환경
- 공통 레이아웃 / 테마 / 반응형 구조

### 2. Market Data

- 코인 목록 조회 서비스
- 차트 시계열 조회 서비스
- UI에서 쓰기 좋은 데이터 정규화 계층

### 3. Dashboard UI

- 종목 검색/목록
- 선택 종목 헤더
- 메인 차트 카드
- 가격 변화, 거래량 등 보조 정보

### 4. Interaction

- 종목 선택
- 시간 범위 변경
- 새로고침 또는 재조회 흐름

### 5. Verification

- 빌드 검증
- 기본 lint 또는 타입 검증
- PR 생성 후 리뷰 확인

## 커밋 전략

1. 앱 스캐폴드와 개발 환경
2. 시장 데이터 계층
3. 대시보드 UI 1차
4. 상태 처리와 마감 다듬기

## 리뷰 전략

- 기능 단위 커밋 후 PR 생성
- Copilot 리뷰와 내부 코드 리뷰 관점으로 회귀/구조 위험 확인
- 필요 시 마지막 커밋에서 리팩터링 반영
